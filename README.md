# Sales-and-Retail-Dashboard

An interactive Power BI dashboard built using the SuperStore dataset to analyze sales performance, customer purchasing behavior, profitability, and future sales trends. The project combines business intelligence with time-series forecasting to help stakeholders make data-driven decisions.

# Project Highlights

•	Developed a dynamic sales dashboard with interactive filters and slicers.
•	Analyzed key business metrics including Sales, Profit, Orders, and Shipping Performance.
•	Visualized sales trends across regions, states, categories, sub-categories, and customer segments.
•	Created a 15-day sales forecast using Power BI's built-in forecasting feature to predict future sales performance.
•	Designed a clean and responsive dashboard for easy business insights.

•	Identified the highest-performing states and regions based on sales and profit.
